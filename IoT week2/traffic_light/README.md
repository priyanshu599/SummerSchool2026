# Traffic Light Controller

## Description

This project simulates a traffic signal using three LEDs.

### Features

- Red LED ON for 5 seconds
- Yellow LED ON for 2 seconds
- Green LED ON for 4 seconds
- Pedestrian button immediately switches to RED for 8 seconds
- Current traffic light state is displayed on the Serial Monitor with timestamp

## Components

- Arduino UNO
- Red LED
- Yellow LED
- Green LED
- Push Button
- 220Ω Resistors
- Breadboard
