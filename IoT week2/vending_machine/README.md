# Vending Machine State Machine

## Description

This project simulates a vending machine using Arduino UNO and implements a finite state machine (FSM).

## States

- IDLE
- COIN_INSERTED
- ITEM_SELECTED
- DISPENSING

## Components

- Arduino UNO
- 3 Push Buttons
- 3 LEDs
- Breadboard
- Jumper Wires

## Features

- Coin insertion
- Item selection
- Cancel option
- LED indication for each state
- Serial Monitor state transition logs
