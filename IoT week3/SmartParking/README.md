# Smart Street Light

## Description

An automatic street light using an LDR and PIR sensor.

## Features

- Detects day and night using an LDR.
- Turns the LED OFF during daylight.
- Turns the LED ON at full brightness when motion is detected at night.
- Dims the LED to 20% brightness after 30 seconds without motion.
- Logs all events on the Serial Monitor with timestamps.

## Components

- Arduino UNO
- LDR Module
- PIR Sensor
- LED
- 220Ω Resistor
