# Ultrasonic Parking Sensor

## Description

This project uses an HC-SR04 ultrasonic sensor to measure the distance of nearby objects and provides parking assistance using LEDs and a buzzer.

## Features

- SAFE indication (>50 cm)
- Yellow warning (20–50 cm)
- Red warning (10–20 cm)
- Emergency alert (<10 cm)
- Non-blocking timing using millis()

## Components

- Arduino UNO
- HC-SR04
- Green LED
- Yellow LED
- Red LED
- Active Buzzer
