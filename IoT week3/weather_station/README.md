# Environmental Monitoring Station

## Description

This project measures temperature and humidity using a DHT22 sensor and Arduino UNO.

## Features

- Temperature in Celsius
- Temperature in Fahrenheit
- Humidity Monitoring
- CSV Output every 2 seconds
- Automatic LED Status Indication

## Components

- Arduino UNO
- DHT22 Sensor
- Red LED
- Green LED
- Breadboard
- Jumper Wires

## Library Used

- DHT sensor library by Adafruit
- Version: 1.4.x
